# 🚀 Developer Portfolio

A modern, responsive, and animated personal developer portfolio built with [Next.js](https://nextjs.org/), [React](https://reactjs.org/), [Tailwind CSS](https://tailwindcss.com/), and [Framer Motion](https://www.framer.com/motion/).

## 🌟 Features

- **Modern Tech Stack**: Built with Next.js 13 and React 18.
- **Beautiful Styling**: Utilizing Tailwind CSS for a sleek, responsive design.
- **Smooth Animations**: Powered by Framer Motion for engaging page transitions and interactions.
- **Component-Driven**: Organized into reusable, clean React components.
- **Production Ready**: Completely optimized for performance and ready to deploy.

## 🛠️ Technologies Used

- [Next.js](https://nextjs.org/)
- [React](https://reactjs.org/)
- [Tailwind CSS](https://tailwindcss.com/)
- [Framer Motion](https://www.framer.com/motion/)

## 🚀 Getting Started

Follow these instructions to get a local copy up and running.

### Prerequisites

Make sure you have [Node.js](https://nodejs.org/) installed.

### Installation

1. Clone the repository
   ```bash
   git clone https://github.com/alarosai/portfolioALR.git
   ```
2. Navigate to the project directory
   ```bash
   cd portfolioALR
   ```
3. Install NPM packages
   ```bash
   npm install
   ```

### Running Locally

Start the local development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the portfolio. You can edit the main page by modifying `src/pages/index.js`. The page auto-updates as you edit the file.

## ☁️ Deployment

This project is highly optimized to be hosted on [Vercel](https://vercel.com/), the easiest and fastest way to deploy Next.js applications.

### Deploying on Vercel

1. Push your code to your GitHub repository.
2. Navigate to your [Vercel Dashboard](https://vercel.com/dashboard) and log in.
3. Click "Add New" and select "Project".
4. Import your `portfolioALR` GitHub repository.
5. Vercel will automatically detect that this is a Next.js app and pre-fill the build settings (`npm run build`).
6. Click **Deploy**.

Vercel will give you a live production URL instantly once the build completes!
